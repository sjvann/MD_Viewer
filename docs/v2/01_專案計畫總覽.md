# 專案計畫總覽

## 專案資訊

- **專案名稱**：Markdown 離線預覽器 (MD_Viewer)
- **技術架構**：.NET MAUI (.NET 9/10) + C# + TypeScript
- **目標平台**：Windows、macOS、Android
- **專案類型**：桌面/行動應用程式
- **預估總工時**：25-35 個工作天
- **預估完成時間**：6-8 週（假設每週 5 個工作天）

## 專案目標

### 主要目標
1. 開發一個功能完整的 Markdown 離線預覽器
2. 支援多平台（Windows、macOS、Android）
3. 提供檔案瀏覽、預覽、編輯、匯出功能
4. 可上架至各平台商店

### 成功標準
- ✅ Windows 平台 MVP 功能完整運作
- ✅ 支援 Markdown 預覽與編輯
- ✅ 支援至少 HTML 和 PDF 匯出
- ✅ 程式碼符合架構設計規範
- ✅ 通過基本功能測試

## 風險管理

### 高風險項目
1. **Android 檔案存取限制** - 需使用 SAF，可能影響開發時程
2. **macOS 沙盒限制** - App Store 上架需特殊處理
3. **TypeScript 與 C# 整合** - JavaScript Interop 可能遇到相容性問題

### 風險緩解策略
- 提前進行技術驗證（Spike）
- 優先實作 Windows 版本作為參考
- 預留緩衝時間處理平台特定問題
