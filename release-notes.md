## MD Viewer v1.0.0

### First Official Release

A cross-platform Markdown viewer and editor built with .NET MAUI.

---

### Download

Download `MD_Viewer-v1.0.0-win-x64.zip` (~100 MB) - Windows 64-bit standalone version (portable)

### How to Use
1. Download and extract the ZIP file
2. Run `MD_Viewer.exe`
3. **No .NET Runtime installation required**

---

### System Requirements
- Windows 10 version 1903 (Build 19041) or later
- x64 processor

---

### Features

- **Markdown Preview** - Supports GitHub Flavored Markdown, tables, code blocks
- **13 Themes** - One Dark Pro, Dracula, Nord, Monokai, Solarized, etc.
- **Edit Mode** - Split view with live preview
- **Export** - PDF and HTML formats
- **File Browser** - Built-in tree directory browser
- **Save Edits** - Save modified Markdown files directly
